# bandits
Python library for Multi-Armed Bandits

Implements the following algorithms:
#### Stochastic
* Epsilon-Greedy
* UCB1
* Softmax
#### Bayesian
* Thompson Sampling
  * Bernoulli, Binomial <=> Beta Distributions
